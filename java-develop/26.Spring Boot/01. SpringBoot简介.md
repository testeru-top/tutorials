## 设计初衷
````ad-note
- [<] 为Spring开发者提供一种，更快速、体验更好的Spring应用开发方式
- [<] 开箱即用，同时也可快速扩展
- [<] 嵌入式的Tomcat
- [<] 绝对没有冗余代码，无需XML配置
````
