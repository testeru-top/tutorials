# Post请求相关注解
- `@RequestMapping`
- `@PostMapping`
- `@RequestBody`
- `@PathVariable`
- `@RequestParam`

## @PostMapping
- 简化常用的HTTP方法的映射，并更好地表达被注解方法的语义
- 相当于`@RequestMapping(method = RequestMethod.POST)`

同 GetMapping 
## @RequestBody

#### required
是否必传参数校验


## 总结
类上的请求注解：
- `RequestMapping`
方法上的请求注解：
- `RequestMapping`
- `POSTMapping`
- `GetMapping`
>还有PUT、DELETE
